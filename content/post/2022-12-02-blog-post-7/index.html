From this, we expected to get more of a sense of which variablesa are highly related to the population of international students.

## Modeling Using the pruned features, for a baseline, we will create linear regression(lm) or generalized linear models(glm).

Then, we are interested in upscaling the performance using methods such as step-wise regression and lasso regression.

Note: We have created lm and glms before, but only naively selected the independent variables.

## Evaluation Since we are solving a regression problem, we will be using Root Mean Square Error as an evaluation method. Additionally, we plan to use information criterion methods (AIC, BIC) to also measure model performance.
